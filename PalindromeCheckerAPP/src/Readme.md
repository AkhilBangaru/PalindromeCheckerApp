# UC6 – Queue + Stack Based Palindrome Check

## Goal

Check whether a string is a palindrome using both Queue and Stack data structures.

## Flow

1. Characters of the string are pushed into a stack and enqueued into a queue.
2. Characters are popped from the stack and removed from the queue.
3. The values are compared.
4. If all characters match, the string is a palindrome.

## Concepts Used

* Stack (LIFO)
* Queue (FIFO)
* Push, Pop
* Enqueue, Dequeue
* Loop
* Console Output

## Sample Output

```
Input text: madam
Is it a palindrome? : true
```
